将提供的 tex 文件中的内容按以下格式拆解成 json 并存储在 chapter.json 中。正文内容只包含 text, image 和 code 三种类型。

{
    "id": "<start-from-1>",
    "title": "<title-of-the-chapter>",
    "sections": [
        {
            "id": "<chapter-id>-<section-id-start-from-1>",
            "title": "<title-of-the-section>",
            "contents": [
                {
                    "type": "text",
                    "content": "<content>"
                },
                {
                    "type": "image",
                    "cloudPath": "<image-path>"
                },
                {
                    "type": "code",
                    "content": "<code>"
                }
            ]
        }
    ]
}
